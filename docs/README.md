# 文档目录

本文档目录包含项目的各种文档和指南。

## 📚 文档列表

### 项目文档 (根目录)

- **[CHANGELOG.md](../CHANGELOG.md)** - 版本更新日志和修改历史

## 📋 文档作用说明

| 文档文件 | 作用 | 适用对象 | 位置 |
|----------|------|----------|------|
| README.md | 项目介绍、快速开始、功能概览 | 所有用户 | 根目录 |
| CHANGELOG.md | 版本更新记录、错误修复历史 | 开发者 | 根目录 |
| getting-started.md | 详细的本地运行指南 | 新手开发者 | docs/ |
| project-analysis.md | 项目架构分析、技术栈评估 | 项目经理 | docs/ |
| project-status.md | 当前功能完成度、待开发功能 | 开发者 | docs/ |
| issue-resolution.md | 问题排查和解决方案 | 维护者 | docs/ |
| test-report.md | 测试结果和质量报告 | QA团队 | docs/ |

## 🎯 使用建议

1. **首次使用**: 从 README.md 开始
2. **开发环境搭建**: 参考 getting-started.md
3. **功能开发**: 查看 project-status.md 了解开发进度
4. **问题排查**: 参考 issue-resolution.md
5. **版本发布**: 更新 CHANGELOG.md

## 📝 文档维护

- README.md: 保持简洁，突出重点
- CHANGELOG.md: 每次发布时更新
- 其他文档: 根据需要更新，保持同步
