## 🚀 正确的启动方法

### 方法1: 手动启动 (推荐)
```bash
# 终端1: 启动后端
python backend/main.py

# 终端2: 启动前端
cd frontend
npm run dev
```

### 方法2: 使用启动脚本
```bash
# 运行一键启动脚本
./start.ps1
# 或
powershell -ExecutionPolicy Bypass -File start.ps1
```

## 🔍 问题排查

### 如果后端无法访问:
1. 确认从项目根目录运行: `python backend/main.py`
2. 检查端口占用: `netstat -ano | findstr :8000`
3. 查看错误信息

### 如果前端无法访问:
1. 确认在frontend目录运行: `cd frontend; npm run dev`
2. 查看终端输出的实际地址
3. 检查防火墙设置
