# 在线大学生选课系统 (Student Course Selection System)

## 项目概述
这是一个基于FastAPI + Vue.js的在线大学生选课系统，支持学生选课、好友系统、转账、消息等功能。

## 技术栈
- **后端**: FastAPI + Python 3.8+ + MySQL (命令行CRUD)
- **前端**: Vue.js 3 + Node.js + Element Plus
- **数据库**: MySQL 8.0+
- **工具**: uvicorn, axios, mysql-connector-python

## 功能特性
### 核心功能
- 👤 用户管理（学生、管理员）
- 📚 课程管理（课程录入、选课、成绩管理）
- 🏫 院系专业管理
- 👥 好友系统（添加好友、好友推荐）
- 💰 转账功能（好友间转账、转账报警）
- 💬 消息系统（好友聊天、消息记录）
- 📊 统计分析（登录记录、选课统计）

### 高级功能
- 🤖 好友推荐算法
- ⚠️ 转账风险预警
- 📈 数据可视化
- 🔐 安全防护（SQL注入防护、权限控制）

## 数据库设计

### 核心表结构
1. **students** - 学生信息表
2. **courses** - 课程信息表  
3. **departments** - 院系信息表
4. **enrollments** - 选课记录表
5. **friendships** - 好友关系表
6. **transactions** - 转账记录表
7. **messages** - 消息记录表
8. **login_logs** - 登录记录表

## 快速开始

### 后端启动
```bash
cd backend
pip install -r requirements.txt
uvicorn main:app --reload --port 8000
```

### 前端启动
```bash
cd frontend
npm install
npm run dev
```

### 数据库初始化
```bash
mysql -u root -p < database/init.sql
```
