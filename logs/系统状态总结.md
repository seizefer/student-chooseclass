### 2. 解决依赖问题的方案

#### 方案A: 解决网络问题（推荐）
1. **检查系统代理设置**
   - Windows设置 → 网络和Internet → 代理
   - 临时关闭代理服务器

2. **清除pip代理配置**
   ```powershell
   pip config unset global.index-url
   pip config unset global.proxy
   ```

3. **使用国内镜像源**
   ```powershell
   pip install -i https://pypi.tuna.tsinghua.edu.cn/simple/ --trusted-host pypi.tuna.tsinghua.edu.cn pydantic-settings python-decouple
   ```

#### 方案B: 手动下载安装
1. 访问 https://pypi.org/ 下载需要的.whl文件
2. 使用 `pip install 文件名.whl` 手动安装

#### 方案C: 使用conda（如果可用）
```powershell
conda install -c conda-forge python-jose passlib
```

### 3. 完整版启动准备

#### 安装完依赖后
```powershell
# 测试完整版配置
python -c "from app.core.config import settings; print('配置正常')"
```
